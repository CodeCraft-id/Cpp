Apa itu C++?

C++ adalah bahasa pemrograman yang dikembangkan oleh Bjarne Stroustrup pada awal 1980-an sebagai perpanjangan dari bahasa C. Bahasa ini dirancang untuk mendukung pemrograman berorientasi objek (OOP), yang memungkinkan pengembang untuk membuat kode yang lebih modular, mudah dipelihara, dan dapat digunakan kembali. Salah satu keunggulan utama C++ adalah kemampuannya untuk menggabungkan pemrograman prosedural dengan pemrograman berorientasi objek, sehingga memberikan fleksibilitas dalam pendekatan pemrograman. Dengan tambahan fitur seperti pengelolaan memori manual, overloading fungsi, dan dukungan untuk template, C++ menjadi pilihan yang kuat untuk pengembangan aplikasi yang membutuhkan kinerja tinggi, seperti perangkat lunak sistem, game, dan aplikasi yang menuntut penggunaan sumber daya yang efisien.


Selain itu, C++ dikenal sebagai bahasa yang sangat efisien dan cepat karena tingkat abstraksinya yang rendah dan kontrol penuh yang diberikan kepada pengembang terhadap perangkat keras. Bahasa ini sering digunakan dalam pengembangan sistem operasi, mesin permainan, dan perangkat lunak tertanam, di mana efisiensi dan performa sangat penting. Meskipun lebih kompleks dibandingkan dengan bahasa pemrograman modern lainnya, seperti Python atau Java, C++ tetap menjadi salah satu bahasa pemrograman yang paling populer dan relevan hingga saat ini, terutama di kalangan pengembang yang membutuhkan kontrol langsung atas memori dan performa aplikasi mereka.

Pengertian Elemen yang ada di C++

int

Tipe data untuk bilangan bulat (misalnya 1, -5, 42).


float

Tipe data untuk bilangan desimal dengan presisi tunggal (misalnya 3.14, -0.01).


double

Tipe data untuk bilangan desimal dengan presisi ganda (misalnya 3.14159, -0.0001).


char

Tipe data untuk satu karakter (misalnya 'a', '7', '%').


bool

Tipe data untuk nilai logika (true atau false).


string

Tipe data untuk rangkaian karakter (misalnya "Halo").


void

Tipe data untuk fungsi yang tidak mengembalikan nilai.


auto

Tipe data variabel ditentukan otomatis berdasarkan inisialisasi.


const

Variabel tidak dapat diubah setelah diinisialisasi.


unsigned

Bilangan bulat hanya bisa bernilai positif.


struct

Mendefinisikan struktur data yang mengelompokkan beberapa variabel.


break

Menghentikan eksekusi dari loop atau switch.


continue

Melanjutkan ke iterasi berikutnya dalam loop.


switch

Struktur kondisional untuk memilih blok kode berdasarkan nilai variabel.


while

Struktur loop yang mengeksekusi kode selama kondisi tertentu terpenuhi.


union

Mengelompokkan beberapa variabel dengan tipe berbeda dalam memori yang sama.


static

Menyimpan variabel sehingga nilainya bertahan antara pemanggilan fungsi, atau membatasi visibilitas variabel ke dalam file.


if & else

Struktur kondisional untuk mengeksekusi kode jika kondisi terpenuhi.


for

Struktur loop dengan kondisi awal, akhir, dan perubahan iterasi.


return

Mengembalikan nilai dari fungsi dan mengakhiri eksekusi fungsi.


extern

Menunjukkan bahwa variabel atau fungsi dideklarasikan di tempat lain.


do

Digunakan dengan `while` untuk loop yang setidaknya dijalankan satu kali.


volatile

Menandakan bahwa nilai variabel bisa berubah di luar kontrol program.


signed

Tipe data yang memungkinkan variabel memiliki nilai negatif atau positif.


typedef

Mendefinisikan alias untuk tipe data.


sizeof

Mengembalikan ukuran dalam byte dari tipe data atau variabel.


register

Menyimpan variabel dalam register CPU untuk akses yang lebih cepat (penggunaan jarang).


goto

Melompat ke label tertentu dalam fungsi (umumnya tidak disarankan).


enum

Tipe data yang memungkinkan variabel memiliki satu dari nilai yang ditentukan.


default

Bagian dari `switch` yang dijalankan jika tidak ada kasus yang cocok.


case

Bagian dari `switch` untuk mendefinisikan blok kode yang dijalankan jika nilai cocok.
